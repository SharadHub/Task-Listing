# Vanilla JS Web App

This is a simple web application built using **HTML**, **CSS**, and **JavaScript (vanilla)**.  
The purpose of this project is to strengthen foundational front-end development skills.

## Features
- Pure HTML, CSS, and JavaScript implementation
- No frameworks or libraries required
- Lightweight and easy to understand

## Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone <repository_url>

2. **Install Live Server extension (if not already installed) in your code editor.**

3. **Open the HTML file in your editor, then right-click and select "Open with Live Server".**

4. **The web app will open in a new window in your default browser.**
