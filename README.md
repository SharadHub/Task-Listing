# To-Do List App

A simple and elegant To-Do List web application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Add Tasks**: Enter a task and click "Add" or press Enter to add it to the list
- **Complete Tasks**: Check the checkbox to mark a task as completed (displays with strikethrough)
- **Delete Tasks**: Remove tasks from the list with the delete button
- **Clean UI**: Modern, responsive design with shadow effects and hover states

## Project Structure

```
to-do-list/
├── to-do.html    # Main HTML file
├── to-do.css     # Stylesheet
├── to-do.js      # JavaScript functionality
└── README.md     # This file
```

## Getting Started

1. Clone or download this repository
2. Open `to-do.html` in your web browser
3. Start adding tasks!

## Usage

1. Type your task in the input field
2. Click the "Add" button or press Enter to add the task
3. Click the checkbox to mark a task as complete
4. Click the "Delete" button to remove a task

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
